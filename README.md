# pantry-pages
a recipe book web app where you can save your favorite recipes, filter them by ingredient, and plan out your meal plan for the week
## Overview
Pantry Pages is a recipe book and meal planner wed application built with html, css and javascript.it helps users
-Save and Organize favorite recipes
-Filter recipes by ingredients
-Plan meals for the wekk using a simple planner interface
My Project shows html, css design and javascript functionality with localStorage across the pages

## Features
1. Semantic html-<header>,<nav>,<main>,<section>,<footer> were used consistently for clarity
2. Recipe Gallery-Browser saved recipes. filter by ingredient, and vieing of details
3. Add Recipe Form- Submits new recipes with validation for needed fields
4. Meal Planner- assign recipes to week slots for each day
5. Responsive Design- CSS Flexbox and media queries ensure usability across laptops, monitors, and phones
6. JS Funtionality -
    - DOM manipulation for dynamic recipe display
    - Event handling for form submission and filtering
    - Form validation with error messages displayed
    - localStorage for recipes and planner data
7. Navigation- Three linked pages(index.html,add.html,planner.html) with consistent styling
